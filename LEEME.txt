# INSTRUCCIONES DE INSTALACIÓN Y EJECUCIÓN

Este proyecto utiliza `uv` para una gestión de paquetes extremadamente rápida y eficiente.

## Requisitos Previos
Tener instalada la herramienta `uv`. Si no la tienes, puedes instalarla con:
`curl -LsSf https://astral.sh/uv/install.sh | sh`

## Pasos para ejecutar el Simulador

1. **Entrar en la carpeta del proyecto:**
   ```bash
   cd control_sim_project
   ```

2. **Ejecutar el script de inicio automático:**
   ```bash
   ./run_sim.sh
   ```

   *Este script se encargará de crear el entorno virtual (.venv), instalar las dependencias (streamlit, control, etc.) y lanzar la aplicación.*

## Instalación Manual (Alternativa)
Si prefieres hacerlo paso a paso:
1. `uv venv`
2. `source .venv/bin/activate`
3. `uv pip install -r requirements.txt`
4. `streamlit run src/app.py`

---
**Estructura del Proyecto:**
- `src/app.py`: Código principal del simulador interactivo web (Streamlit).
- `src/app_escritorio.py`: Código del simulador interactivo de escritorio (PyQt6).
- `report/informe_final.tex`: Código fuente LaTeX del informe final.
- `report/informe_final.pdf`: PDF compilado listo para entregar.
- `scilab/`: Modelos (.zcos) y simulaciones (.sce) de Scilab / Xcos.
- `scripts/`: Scripts de Python para generar gráficos y diagramas de bloques.
- `docs/`: Documentación de apoyo (marco teórico, guías del TFI, etc.).
- `run_sim.sh`: Script de automatización de entorno y ejecución web.
- `run_desktop_sim.sh`: Script de automatización de entorno y ejecución de escritorio (PyQt6).
- `requirements.txt`: Lista de librerías necesarias.
